# Task Manager (Cohort3_Assignment7)

This Project is strictly built with HTML, CSS, and Vanila JS , A fully interactive and Browser Rendering Explorer.

This web serves as a visual playground and documentation sheet for core browser rendering pipelines, DOM manipulation techniques, property-attribute divergence, and event propagation dynamics.


## Key Feature 

Interactive User Dashboard 

 Create , edit , complete, delete tasks.
 Live category of Completed task & pending task.
 Real-time comparison widget showing how the browser manages input.value (DOM property) independently   of input.getAttribute("value") (HTML markup attribute).
 Event Propagation Shown via Nested Structure : Grandparent --> Parent --> Child Button.
 Event Capturing and Event Bubbling phases explanation with example.
 Created Theme toggle option in User Dashboard.



 ## Some Important Topic/Terms Explanation 

 1. Parsing :- The Process of reading HTML code and converting it into a structure (DOM) that the browser can understand and display..
 2. Tokenization :- Tokenization is the process to break the HTML code into small meaningful pieces called Tokens..
 3. DOM Tree :- The DOM is Tree like Structure/representation of an HTML created by the Browser after parsing HTML. Each HTML Element becomes an node Object , so that JavaScript can manipulate..
 4. CSSOM Tree :- The CSSOM is Tree like representaion of CSS created by the Browser after parsing CSS..
 5. Render Tree :- The Render tree is created by combining the DOM and CSSSOM tree..
 6. Event Bubbling :-  Event Bubbling is the default event propagation behaviour in JavaScript Where an event starts from target element and then propagate upward thorugh its parent elements up to the document..
 7. Event Capturing :-  Event Capturing is the even propagation behaviour in Javascript Where an event travels from the document(root) down to the targetted element..
 8. Event Delegation :- Event Delegation is a technique where a single event attached to a parent element to handle events for its child elements using event bubbling..
